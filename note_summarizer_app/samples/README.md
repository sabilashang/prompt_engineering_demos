## üöÄ How to Use:

1. **Open any sample file** in a text editor
2. **Copy the entire content** (Ctrl+A, Ctrl+C)
3. **Start the Note Summarizer app:**
   ```bash
   python main.py
   ```
4. **Open** http://localhost:5002 in your browser
5. **Paste** the copied text into the text area
6. **Select** your preferred summary type:
   - **Concise** - Short, bullet-point summary
   - **Detailed** - Comprehensive overview
   - **Key Points** - Main takeaways only
7. **Click** "Summarize" button
8. **View** your AI-generated summary!
